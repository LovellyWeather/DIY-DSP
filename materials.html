Necessary Materials

This project is based around the Pedal-Pi from ElectroSmash. On this page you'll find information on what all kits, tools, and components you'll need in order to assemble the pedal.

The Pi

For this project you're going to need to order the Raspberry Pi Zero from an online retailer. For this walkthrough we used the Raspberry Pi Zero Starter Kit from CanaKit. Included in this kit is:

  • Raspberry Pi Zero Board
  • 8 Gb Class 10 MicroSD Card (with OS)
  • 1A Power Supply
  • USB A to Mini Adapter
  • Mini HDMI Adapter

Along with this kit you're also going to need a few other things. Each of these can be purchased from Amazon or any other online retailer. You're going to need:

  • HDMI Cable
  • Keyboard and Mouse
  • Monitor or TV
  • Powered USB Hub

The Pedal

Along with the Pi you're going to need to purchase the Pedal Pi from Electrosmash. We recommend purchasing the Pedal-Pi Full Kit from Electrosmash. This kit includes:

  • The PCB
  • Power Adapter
  • Circuit Components
  • Plastic Covering

You're also going to need some soldering supplies. If you already have some then great! If not I would recommend getting a beginners soldering kit that has:

  • Soldering Iron
  • Solder
  • Sponge

If you've never soldered don't worry, we'll go over some basic technique in a future section.
